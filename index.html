Cara update data lowongan (klik untuk lihat)
Data ini statis (di-scrape manual), bukan live dari situsnya. Untuk update:
  1. Login ke maganghub.kemnaker.go.id
  2. Buka halaman Lowongan, buka Console browser (F12 → tab Console)
  3. Ketik allow pasting kalau diminta, lalu jalankan script scraping (minta Claude kirim ulang scriptnya kalau perlu)
  4. Tunggu sampai selesai, file akan otomatis ke-download
  5. Rename file hasil download jadi data.json, lalu upload/timpa ke repo GitHub kamu
